You were previously assigned the task to sort seven identical objects based on their weights. This is very similar to how a computer handles tasks related to the sorting a set of numbers in ascending or descending order.

To perform such a task, a computer applies a sorting algorithm. There are a number of different sorting algorithms which are in use today, each of which have different advantages. Some of them are faster, whereas some of them require less memory to store the information. However, the basic goal of each algorithm is the same - to teach the computer an ordered set of rules according to which it can perform a task, which in this case is sorting of numbers in an order.
