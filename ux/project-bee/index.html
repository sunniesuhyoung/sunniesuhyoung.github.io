AR Muse: Project Bee

Design and development of a web application
for creating mazes for AR game

Overview

My internship was at the Keio-NUS CUTE Centre during the summer of 2015. The AR Muse Project Bee was an augmented reality maze game whose goal was to enhance learning in museums.

My project was to design and develop a web platform which will function as an internal tool and enable the users to create mazes and various maze elements, and incorporate them into the game. I designed the interface and the information architecture of the web platform. I was also responsible for the development of the interface and integrating it with the database for the android application.

The AR Muse: Project Bee is currently live and available for download from Google Play.

Role

I worked in a team of multidisciplinary researchers. My role involved both design and development and I collaborated with one designer and one developer in this project.

Objective

The task was to design and prototype an online maze editor for the AR Muse platform which will be used to create mazes for the augmented reality game. The AR Muse: Project Bee is a maze game which uses several characters and elements. The online maze editor is to provide support to create mazes for the game.

Interactions

The AR Muse: Project Bee has six tile elements. The elements can be given certain properties while creating the maze:

The Wall is an obstacle tile which is drawn by clicking or clicking and dragging along the tiles.
Bee is a playable character which can be drawn only once on the canvas. Any new instance of bee drawn will replace the previous instance. The maze editor allows the user to define the speed, maximum speed and minimum speed of the bee for each game.
Flower is a boost element consuming which will enable the playable character to increase its points gained
Hive is another boost element.
Mite His a hrmful element when in contact with the bee, reduces its speed. The editor allows the user to preset the speed and movement of the mite.
Endpoint another element which can only exist singularly in the maze. This is the goal of the playable character
Eraser is an universal erase function which functions to remove any tile.

Development

The tool for allowing the users to create custom mazes for the AR Muse game was developed using the JavaScript D3 library. The data stored in the database is then used in the game.

My role in the project was in the design of the interface, in defining and developing the interactions with which the user interacts to draw the elements and characters, and storing this information in the database. The final developed interface of the tool is shown in the image below.

About AR Muse

The AR Muse: Project Bee is an augmented reality game which is built on the Vuforia Engine (Unity 3D). Project Bee is one from a series of augmented reality games created by the Augmented Learning Group at the Keio-NUS Cute centre.

Project Bee can be run on a tablet or mobile phone. The game will begin when it detects the AR Muse: Project Bee poster on the background. The user has to maneouvre the bee through the maze, collect the flowers, evade the mites and reach the end-point.
